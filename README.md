
# Demo SQL Query Notebook

## Overview
This notebook demonstrates how to use Databricks SQL queries within a Databricks notebook environment. It showcases querying, transforming, and visualizing data using Spark DataFrames and Databricks SQL syntax.

## Features
- Executing SQL queries using Databricks SQL.
- Loading and displaying data with Spark DataFrames.
- Data transformation and aggregation.
- Visualization using `display()`.

## Usage
1. Open the notebook in Databricks.
2. Run each cell sequentially to execute SQL queries and view results.
3. Modify queries as needed for your own datasets.

## Requirements
- Databricks workspace access.
- Required tables/data sources available in your environment.

## Example
sql
SELECT * FROM your_table LIMIT 10
